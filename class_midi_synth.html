Detailed Description

A monophonic/polyphonic synthesiser base class which can be supplied with a custom voice.

Supports different kinds of after touch, pitch bend, velocity and after touch curves, unison (currently monophonic mode only) NOTE: This is not currently particularly efficient, and needs a bit more work to be more generalisable
